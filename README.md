# PortfolioHUB 🚀
Repositório dedicado à centralização de projetos acadêmicos e profissionais desenvolvidos durante o Bootcamp de Engenharia do CEUB.

## 🌐 Portfólio no Ar (GitHub Pages)
Acesse a página publicada em tempo real:
 https://github.com/Matheusxsouza/PortifolioHub/settings/pages

## 📂 Organização de Diretórios
* `/` : Raiz contendo o arquivo de indexação principal (`index.html`).
* `/documentos` : Diretório contendo o Currículo Profissional em PDF.
* `/slides` : Diretório contendo a Apresentação Executiva em PDF.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Git:** Controle de versionamento local.
* **GitHub:** Hospedagem de código em nuvem.
* **GitHub Pages:** Publicação da página web estática.
